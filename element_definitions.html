Element Definitions

The following terms are outlined in chapters 2 and 3 of our text.

<dl>
The <dl> elements is used for a list of difinitions
<dt>
The <dt> element is used to house the term being defined in the difinition list.
<dd>
The <dd> element is used to house the difinition of the term in the difinition list.
<h1 - h6>
The <h1 - h6> codes are used to display verious heading sizes <h1> being the largest and <h6> being the smallest.
<p>
The <p> element is to store blocks of text to display in a paragraph formate.
<b>
<strong>
The <b> & <strong> elements are both used to add bold to the text being used.
<i>
<em>
The <i> & <em> are used make the text italicized.
<sup>
The <sup> element is used to contain characters that should be superscript such a number to the second power. Exp. 32
<sub>
The <sub> element is used to contain characters that should be subscript like chemical formulas. Exp. CO2
<br />
The <br /> element is used to add a line brake in a group of text.
<hr />
The <hr /> element is used to creratea break between themes in text by adding a horizontal ruel between them.
<blockquote>
The <blockquote> element is used for long quotes that may take up a full paragraph.
<q>
The <q> element is used for shorter quotes.
<abbr>
You would use the <abbr> element to store the meaning of an abbreviated word.
<acronym>
You would use the <acronym> element to store the meaning of an acronym.
<cite>
The <cite> element is used to show where a citation is from.
<dfn>
The <dfn> element is used when defining a new terminology for the 1st time.
<address>
The <address> element holds the contact information for the author of the page.
<ins>
The <ins> element is sued to show infoormation that has been added or inserted into a document.
<del>
<s>
The <del> & <s> elements cane be used to delete information that is no longer needed or accurate.
<u>
The <u> element is used to underline the text. This is no longer used.
<ol>
The <ol> element is used to display a ordered list.
<ul>
The <ul> element is used to display a non-ordered list.
<li>
The <li> element holds each list item.
